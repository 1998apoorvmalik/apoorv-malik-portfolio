Personal Projects

  • Generic Genetic Algorithm:
    Python project for abstract & general implementation of the Genetic Algorithm. It can be used to solve all kinds of relevant problems, such as n-queens, 8-puzzle, 15-puzzle, target string problem, knapsack problem, and all those problems which feature deterministic state change (deterministic transition) and are quantifiable by a reward function.
    Github Link: https://github.com/1998apoorvmalik/generic-genetic-algorithm
  • Chess Game:
    A game of chess entirely made from scratch in Flutter. I have coded everything using no packages, libraries, or plugins - including game logic, game board, NegaMax algorithm-based Al, and online multiplayer.
    Github Link: https://github.com/1998apoorvmalik/flutter-chess
  • Teachable Image Classifier:
    Similar to Google's teachable machine, this project provides a visual interface for training Deep Learning models by allowing users to create classes, add image samples, and set training parameters.
    Github Link: https://github.com/1998apoorvmalik/teachable-image-classifier
  • Sorting Algorithm Visualizer:
    Developed this program with Flutter. It enables us to visualize how complex sorting algorithms (Merge, Heap, Quick, Bubble & Insertion Sort) work.
    Github Link: https://github.com/1998apoorvmalik/sorting-algorithm-visualizer

Professional Projects

  • ASK Fras-Le Friction PVT. LTD:
    Created an application to perform dimensional analysis and classification of brake liners. Multiple vision cameras above the conveyor line calculate the dimensions of an incoming part, which is then further analyzed to classify it as ok or not good (based on some tolerance level).
  • Samsung India:
    Shape and color inspection of SMT nozzles. The software inspected and categorized the incoming part as ok or not good based on specific features. There were 40 different part categories the algorithm had to learn.
  • Minda Group:
    Completed two different projects for this company - developed an OCR (Optical Character Recognition) algorithm for number reading in Automotive Key Tags. And implemented a deep learning-based solution for classification & categorization of wheel rims.
